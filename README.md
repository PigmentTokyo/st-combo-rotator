# Combo Rotator（预设 + 连接 轮换）— SillyTavern 扩展

把「预设（AI 响应配置）」和「连接配置文件（Connection Profile）」配成多个自由命名的组合，
通过悬浮窗按「每 N 轮对话」**顺序**或**完全随机**地轮换。

## 功能
- 悬浮窗（右下角 `⇄` 按钮开/关，窗口可拖动）。
- 自由添加组合，每组 = 1 个预设名 + 1 个连接配置名，名字随便起（AB、CD……）。
- 模式：顺序轮换 / 完全随机。
- 频率：每 N 轮对话切换一次（1 轮 = 你发一条 + AI 回一条）。
- 总开关关闭 = 永不切换。
- 每个组合行可「▶ 立即应用」，另有「应用当前组合 / 重置计数」按钮。
- 状态栏显示当前组合与「距下次切换还剩几轮」。

## 安装
1. 找到 SillyTavern 安装目录下的第三方扩展文件夹：
   ```
   <SillyTavern>/public/scripts/extensions/third-party/
   ```
2. 把整个 `st-combo-rotator` 文件夹复制进去，结构应为：
   ```
   third-party/st-combo-rotator/manifest.json
   third-party/st-combo-rotator/index.js
   third-party/st-combo-rotator/style.css
   ```
3. 刷新 / 重启 SillyTavern。右下角会出现 `⇄` 悬浮按钮。

> 也可以用 SillyTavern 的「扩展 → 安装扩展」从 Git 仓库地址安装；本地复制最简单。

## 前置条件（重要）
本扩展靠两个**内置 slash 命令**生效，请先确认它们能用：
- `/profile <名字>` —— 切换连接配置文件。由内置的 **Connection Manager** 提供，需要你先在
  「API 连接 → 连接配置文件」里建好若干 profile 并**命名**。
- `/preset <名字>` —— 切换当前 API 的预设（AI 响应配置）。

组合行里的「预设」和「连接配置」都是**下拉菜单**，自动列出你当前已有的全部预设和连接配置文件，
直接选即可。如果想用列表里没有的名字，选下拉里的「✎ 手动输入…」自己打。
新建/删除了预设或连接配置后，点面板里的「↻ 刷新选项」即可重新拉取列表。

## 用法
1. 点 `⇄` 打开悬浮窗。
2. 「+ 添加组合」，每行填：组合名、预设名、连接配置名。至少加 2 组才会轮换。
3. 选模式（顺序/随机），设频率（每几轮切一次）。
4. 勾「启用轮换」。之后每完成相应轮数，下一轮自动切到新组合，并弹提示。
5. 想立刻用某组合，点该行的 `▶`。

## 切换时机说明
计数在**每条 AI 回复完成后**累加；达到频率时切换组合，因此**新组合从下一轮生效**，
不会打断正在进行的本轮生成。切换聊天会自动把计数清零。

## 注意
- 顺序应用是「先 `/profile` 再 `/preset`」，避免连接配置自带的预设把你指定的预设覆盖掉。
- 如果某些连接配置文件本身就绑定了预设，确保组合里仍显式写上你要的预设名，以它为准。
- 报错「应用组合失败」一般是名字对不上，核对预设/连接配置名拼写即可。
