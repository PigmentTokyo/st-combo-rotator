# Combo Rotator（预设 + 连接 轮换）— SillyTavern 扩展

把「预设（AI 响应配置）」和「连接配置文件（Connection Profile）」配成多个自由命名的组合，
通过悬浮窗按「每 N 轮对话」**顺序**或**完全随机**地轮换。

## 功能
- 悬浮窗（右下角 `⇄` 按钮开/关，窗口可拖动）。
- 自由添加组合，每组 = 1 个预设名 + 1 个连接配置名，名字随便起（AB、CD……）。
- 模式：顺序轮换 / 完全随机。
- 频率：每 N 轮对话切换一次（1 轮 = 你发一条 + AI 回一条）。
- 总开关关闭 = 永不切换。
- 每个组合行可「▶ 立即应用」，另有「应用当前组合 / 重置计数」按钮。
- 状态栏显示当前组合与「距下次切换还剩几轮」。

## 安装
1. 找到 SillyTavern 安装目录下的第三方扩展文件夹：
   ```
   <SillyTavern>/public/scripts/extensions/third-party/
   ```
2. 把整个 `st-combo-rotator` 文件夹复制进去，结构应为：
   ```
   third-party/st-combo-rotator/manifest.json
   third-party/st-combo-rotator/index.js
   third-party/st-combo-rotator/style.css
   ```
3. 刷新 / 重启 SillyTavern。右下角会出现 `⇄` 悬浮按钮。

> 也可以用 SillyTavern 的「扩展 → 安装扩展」从 Git 仓库地址安装；本地复制最简单。

## 前置条件（重要）
本扩展靠两个**内置 slash 命令**生效，请先确认它们能用：
- `/profile <名字>` —— 切换连接配置文件。由内置的 **Connection Manager** 提供，需要你先在
  「API 连接 → 连接配置文件」里建好若干 profile 并**命名**。
- `/preset <名字>` —— 切换当前 API 的预设（AI 响应配置）。

组合行里的「连接配置」是**下拉菜单**，自动列出你当前已有的全部连接配置文件，直接选即可。
预设和模型都打包在连接配置里，所以扩展只切连接配置，不单独切预设。
想用列表里没有的名字，选下拉里的「✎ 手动输入…」自己打；新建/删除连接配置后，点「↻ 刷新选项」重新拉取。

## 推荐用法：用「连接配置」绑定模型（一份预设搞定）
**预设本身不保存模型**，所以靠存两份预设是绑不住模型的。要让「A 版用模型A、B 版用模型B」，
应该用**连接配置文件**来绑模型：

1. 在 SillyTavern「API 连接 → 连接配置文件 → 新建」里，建 **A 版**，勾选：
   - ✅ API（如 claude）
   - ✅ Settings Preset（指向你那份预设）
   - ✅ **模型** ← 选模型A（关键！这一步才是真正绑模型）
   - 其它（Proxy / Regex / Reasoning 等）想固定就勾，否则不勾
   - 起个清楚的名字，如 `剧本-OpusA`
2. 再建 **B 版**，勾选完全相同，只把**模型**改成模型B，名字如 `剧本-SonnetB`。
3. 回到本扩展：每个组合**只选「连接配置」**（A 版 / B 版）即可——
   连接配置切过去时已经把预设+模型一起设好了，扩展不需要也没有单独的预设选项。

> 只有当你想给两个模型用**不同采样参数**时，才需要两份预设：建两份预设，并在各自连接配置里
> 把 Settings Preset 指向对应那份。

### 不想看到「是否刷新页面以切换正则」的提示
有两种来源：

1. **连接配置带了 Regex Preset**：建连接配置时不勾 Regex Preset 即可。
2. **预设(Settings Preset)内部夹带了启用的正则**（提示写「预设…包含被启用的正则」）：
   如果两份预设启用的条目不同、必须各自加载，这个提示就避不开——但它只是用来把正则
   **追溯到旧消息**，对**新生成的消息正则照常生效**。所以面板里提供了开关
   **「隐藏切换时的『正则需重载』提示」**（默认开），切换时自动把这条 toast 吞掉，正常往下聊即可。
   若偶尔想把正则补用到旧消息，关掉该开关、按提示点一次重载即可。

## 用法
1. 点 `⇄` 打开悬浮窗。
2. 「+ 添加组合」，每行：组合名 + 连接配置。至少加 2 组才会轮换。
3. 选模式（顺序/随机），设频率（每几轮切一次）。
4. 勾「启用轮换」。之后每完成相应轮数，下一轮自动切到新组合，并弹提示。
5. 想立刻切到某组合（一键切换本次用哪个模型+预设），点该行的 `▶`。

## 切换时机说明
计数在**每条 AI 回复完成后**累加；达到频率时切换组合，因此**新组合从下一轮生效**，
不会打断正在进行的本轮生成。切换聊天会自动把计数清零。

## 注意
- 切换时只执行 `/profile <名字>`，预设、模型、代理等都由连接配置本身携带，所以请确保
  连接配置里勾了你需要带的项（至少 API + Settings Preset + 模型；不同后端再加 Proxy Preset）。
- 报错「应用组合失败」一般是名字对不上，核对连接配置名拼写即可。
